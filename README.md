
# Crypto Arbitrage Bot PRO+

Advanced multi‑exchange crypto arbitrage scanner.

Supported exchanges:
- Bybit
- OKX
- MEXC
- HTX

Main Features
-------------
• Async scanning of the entire market
• Spread detection between exchanges
• Liquidity / orderbook depth analysis
• Withdraw availability check
• Fee and profit estimation
• Fake volume filter
• Network compatibility check (simplified)
• Top‑5 best opportunities
• Telegram alerts (optional)
• Modular architecture for adding exchanges

Install
-------

pip install -r requirements.txt

Run
---

python main.py
